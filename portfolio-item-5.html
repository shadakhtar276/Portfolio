Creating a News Website Using JavaScript And News API

In this Project, This website will use news API. The News API is a simple and easy-to-use API that returns JSON metadata for headlines and articles live all over the web right now. This website will show live news to the user, which will be displayed with the help of news API.

To get the News API:

  • Click on https://newsapi.org/
  • Click on “ Get API Key”
  • Register for API Key
