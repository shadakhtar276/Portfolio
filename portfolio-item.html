Corona Map

Plotting CoronaVirus Cases On World Map - Interactive Dashboard Using JavaScript & MapBox

COVID-19, or the coronavirus, is currently making its way across the globe. This project is plotting CoronaVirus cases on World Map intractive dashboard using Javascript, MapBox , HTML5 & CSS3

So this visualation Shows us how CoronaVirus is spreading across the world. In this the different color shades of markers shows the different number of cases.

The red marker highlights the highest number of cases while the color of marker starts changing as the number of cases decreased. The black color marker denotes the lowest number of cases. In this Fetch API is used to Fetch the data of CoronaVirus cases.
